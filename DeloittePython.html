Deloitte

Definity Insurance is a Canadian Property & Casualty insurance company offering automobile, property, liability, agriculture, and surety insurance. This project involved building scalable, modern web applications that streamline operations and enhance digital experiences for both internal users and customers.
Built and maintained scalable full-stack applications using Django, Flask, and React, improving both server-side processing and client-side interactivity. This led to a 30% reduction in page load times and enhanced the overall user experience for internal users and customers.
Designed and implemented RESTful APIs that integrated seamlessly with MySQL and MongoDB databases, allowing real-time data transactions and boosting backend performance by over 40% during peak usage periods.
Spearheaded the containerization of microservices using Docker and deployed them using Kubernetes on AWS EKS, achieving high availability and 99.9% system uptime across multiple environments.
Automated the CI/CD pipeline using GitHub Actions and AWS CodeDeploy, enabling continuous integration and reducing manual intervention. This improved the release cycle speed by 35% and increased deployment reliability.
Provisioned and managed robust AWS cloud infrastructure, including services like EC2, S3, Lambda, IAM, RDS, and CloudWatch. This ensured scalable deployments aligned with security and compliance best practices.
Developed real-time Kafka data pipelines to facilitate high-throughput event streaming between microservices and external systems. This helped eliminate batch processing delays by 40% and supported real-time analytics use cases.
Worked closely with DevOps and engineering teams to implement Infrastructure as Code (IaC) practices using Terraform and Ansible, which significantly accelerated cloud provisioning and improved deployment consistency.
Led the integration of third-party APIs and implemented automated testing workflows using PyTest. This reduced the defect rate in production deployments by 30% and improved the overall reliability of the microservices ecosystem.
